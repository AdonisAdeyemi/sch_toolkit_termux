🧩 5. SERVICE LAYER (CORE ENGINE)

📄 src/ReportCardService.php

This is your brain.

Responsibilities:

✔ Loop students
✔ Render HTML template
✔ Pass to DomPDF
✔ Output PDF

Structure:

class ReportCardService
{
    public function renderAll($students)
    {
        $html = "";

        foreach ($students as $student) {
            $html .= $this->renderStudent($student);
        }

        return $html;
    }

    public function renderStudent($student)
    {
        ob_start();
        include "../templates/report_card.php";
        return ob_get_clean();
    }
}

📦 8. PDF GENERATION FILE

📄 public/generate.php

This is the final pipeline:

use Dompdf\Dompdf;

$service = new ReportCardService();
$html = $service->renderAll($data);

$dompdf = new Dompdf();
$dompdf->loadHtml($html);
$dompdf->setPaper('A4', 'portrait');
$dompdf->render();

$dompdf->stream("reportcards.pdf", ["Attachment" => 0]);

📏 9. DESIGN RULES (CRITICAL FOR DOMPDF)

MUST FOLLOW:

✔ Tables for layout
✔ Inline styles preferred
✔ No flex/grid
✔ No external CSS dependencies
✔ Avoid floats
✔ Avoid absolute positioning (except watermark)


---

📄 10. A4 SETTINGS (IMPORTANT)

Always assume:

A4 = 210mm × 297mm

DomPDF:

$dompdf->setPaper('A4', 'portrait');
